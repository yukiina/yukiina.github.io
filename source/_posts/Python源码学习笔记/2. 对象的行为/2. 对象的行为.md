### 一. 对象的创建

​	在Python中创建一个内置对象通常有两种方式, 一种直接使用赋值语句, 解释器会自动推导对象的类型, 一种是和自定义类型一样使用`类型()`的方式创建。

```python
# 直接赋值方式创建对象
var1 = 1.1
var2 = [1, 2, 3]
var3 = {'A': 1}

# 使用 类型() 方式
var1 = float(1.1)
var2 = list()
var3 = dict()
var4 = MyClass()
```
